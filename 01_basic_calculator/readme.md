# Basic Calculator

## Description
A simple command-line calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User input handling

## Technologies Used
- Python

## How to Run
python main.py

## Example
Enter first number: 10  
Enter operator: +  
Enter second number: 5  
Result: 15
